Slack的po文功能實在是很多缺點，不支援markdown、無法即時瀏覽、沒有tag/分類常常找不到，code highlight只有一種，原則上只適合po短篇的文章，之後會把slack上現存的文章搬過來這裡。  

底下列出幾個在這裡po文很nice的功能
<!--more-->
- code highlight

{% highlight python %}
>>> colors = ['red', 'green', 'blue']
>>> vals = [55, 89, 144, 233]
>>> for col, val in zip(colors, vals):
...     print(col, val)
...
('red', 55)
('green', 89)
('blue', 144)
{% endhighlight %}
